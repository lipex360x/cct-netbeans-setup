# cct-netbeans-setup

Python automation to configure NetBeans projects for academic classes.

Targets Java projects built with Ant that require JUnit test setup.
Designed to be invoked via `uv run <script-url>`.

## Requirements

- Python 3.11+
- [uv](https://docs.astral.sh/uv/)

## Usage

```bash
uv run <script-url>
```
